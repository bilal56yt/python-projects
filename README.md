# 🐍 Python Projects — Muhammad Bilal

A collection of Python scripts, automation tools, and backend mini-projects.

## Projects Included

### 1. number_guess.py
A fun number guessing game where the computer picks a random number and the user guesses it.

### 2. password_generator.py
Generates strong random passwords with letters, numbers and symbols.

### 3. todo_cli.py
Simple command-line To-Do list manager — add, view, and delete tasks.

### 4. web_scraper.py
Basic web scraper using `requests` and `BeautifulSoup` to extract page titles.

## How to Run
```bash
# Make sure Python 3 is installed
python number_guess.py
python password_generator.py
python todo_cli.py
```

## Requirements
```bash
pip install requests beautifulsoup4
```
